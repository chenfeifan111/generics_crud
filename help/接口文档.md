## POST 查询

POST /users/query

> Body 请求参数

```json
{
  "page": {
    "page_num": 1,
    "page_size": 10
  },
  "filters": {
    "age": 18,
    "name": "zs"
  },
  "orders": {
    "age": "asc",
    "id": "desc"
  }
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 是 |none|

> 返回示例

> 200 Response

```json
{
  "code": 200,
  "msg": "success",
  "data": [
    {
      "id": "3",
      "name": "zss",
      "age": 21
    }
  ],
  "total": 0,
  "page": {
    "page_num": 2,
    "page_size": 1
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|
